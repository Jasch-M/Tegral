# TW Greeter

Tegral Web Greeter is a simple feature that prints a greeting message, including the Tegral version, just before all other services start.

<PackageVersions libraries={[
    { name: 'tegral-web-greeter', catalog: 'web.greeter' }
]}/>


## Hide the message

You can hide the welcome message using [Tegral Logging](/modules/core/logging/index.mdx) to disable the logger:

```toml title="/tegral.toml"
[tegral.logging.loggers."tegral.greeter"]
level = "Off"
```
